nodEdit — Minimal IDE

2014-03-20

Minimal IDE dedicated to single page application (SPA) development under NodeJS with WebKit.
This simple tool implements the read - execute - print - loop (REPL) paradigme using only HTML5, CSS3 and Javascript to develop/debug Shell scripts, NodeJS code or Markdown documentation files.

CodeMirror-3.22

Download from GitHub repository.
The project page is http://codemirror.net
The manual is at http://codemirror.net/doc/manual.html

Editor Mode and Theme are selected according to the extension of the loaded file — as expected, the default mime type is text/plain. Some functions like jshit are automatically activated when the current file type is Javascript. Shortcut [Ctrl-SPACE] triggers code completion based on the words found around in the present document (not the language vocabulary).

Built in keyboard commands

Extract from Source code with additional nodEdit specific commands.

WARN: Note that [Cmd-Q] key press from the Front-Page kills the OS-X process without saving the lastContext. Then use preferably the tray-menu Quit command in order to correctly record context for later use — you may also use Cmd-Q but only from the Editor-Page.

System commands only available from Front-Page:

  • “Ctrl-SPACE”: “Recall last context”,
  • “Ctrl-ENTER”: “Toggle browser debug window”,
  • “Ctrl-O”: “Open file chooser”,
  • “Ctrl-N”: “Create a new document”,
  • “Ctrl-L”: “Reload browser”,

General Editor-Page commands:

Text edit commands:

  • “Cmd-C”: “Copy”,
  • “Cmd-X”: “Cut”,
  • “Cmd-V”: “Paste”,
  • “Cmd-A”: “selectAll”,
  • “Cmd-D”: “deleteLine”,
  • “Cmd-Z”: “Undo”,
  • “Shift-Cmd-Z”: “Redo”,
  • “Cmd-Y”: “Redo”, (yank)
  • “Ctrl-Space”: “autocomplete”, (agnostic)

Move cursor:

  • “Cmd-Up”: “goDocStart”,
  • “Cmd-End”: “goDocEnd”,
  • “Cmd-Down”: “goDocEnd”,
  • “Alt-Left”: “goWordLeft”,
  • “Alt-Right”: “goWordRight”,
  • “Cmd-Left”: “goLineStart”,
  • “Cmd-Right”: “goLineEnd”,

Delete:

  • “Alt-Backspace”: “delWordBefore”,
  • “Ctrl-Alt-Backspace”: “delWordAfter”,
  • “Alt-Delete”: “delWordAfter”,

File commands:

  • “Cmd-O”: “Open”,
  • “Cmd-N”: “New”,
  • “Cmd-S”: “Save”,
  • “Shift-Cmd-S”: “SaveAs...”,

Find and replace:

  • “Cmd-F”: “find”,
  • “Cmd-G”: “findNext”,
  • “Shift-Cmd-G”: “findPrev”,
  • “Cmd-Alt-F”: “replace”,
  • “Shift-Cmd-Alt-F”: “replaceAll”,

Indent line or simply use Tab chars:

  • “Cmd-[”: “indentLess”,
  • “Cmd-]”: “indentMore”

HTML Emmet (formerly ZenCoding) command:

  • “Cmd-E”: “expand abbrev”,
