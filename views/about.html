nodEdit — Minimal IDE

2014-03-31

Minimal IDE dedicated to single page application (SPA) development under NodeJS with WebKit. This simple tool implements the read - execute - print - loop (REPL) paradigme using only HTML5, CSS3 and Javascript to develop/debug Shell and Python scripts, HTML and NodeJS code as well as Markdown or plain text documentation files.

Rationales

The goal of this development was to provide just what you need and no more, avoiding clobbered graphic user interface (no Eclipse like GUI) and nearly useless exotic functions (like FTP and SVN interface). Don't forget UNIX golden rule as in the early 70's and keep in mind that “small is beautiful..!”

Getting Started

On launch, the so called Front-Page is displayed. From there, Drag & Drop a file into the page footer to open it (or press Ctrl-O to use a file browser). Now you can edit the file then use buttons in the page header to Save / Run / Preview its content or result according to the document type: Script / HTML or Markdown.

Beyond this basic editor usage, nodEdit provides some additional functions.
To be mentioned:

  • Javascript list of functions and variables with their associated comments (a la JSDoc) with goto line number on click
  • Markdown formatting keyboard shortcuts (or menu entries) for Bold, Italic, Web link, List, etc…

Built in keyboard commands

Extract from Source code with additional nodEdit specific commands.

WARN: Note that [Cmd-Q] key press from the Front-Page kills the OS-X process without saving the lastContext. Then use preferably the tray-menu Quit command in order to correctly record context for later use — you may also use Cmd-Q but only from the Editor-Page.

Key stroke→ Actions only available from Front-Page
Ctrl-SPACE → Recall last editor context
Ctrl-W → Toggle browser debug window
Ctrl-O → Open file chooser
Ctrl-N → Create a new document as text/plain
Ctrl-L → Reload WebKit browser

General Editor-Page commands:

Text edit commands:

  • “Cmd-C”: “Copy”,
  • “Cmd-X”: “Cut”,
  • “Cmd-V”: “Paste”,
  • “Cmd-A”: “selectAll”,
  • “Cmd-Z”: “Undo”,
  • “Shift-Cmd-Z”: “Redo”,
  • “Cmd-Alt-O”: “Toggle overwrite mode”,
  • “Ctrl-T”: “Toggle comment line(s)”,
  • “Ctrl-Space”: “autocomplete” (agnostic)

Click inside left part of the gutter to select entire line(s) staring from the one where is the current cursor.

Move cursor:

  • “Cmd-Up”: “goDocStart”,
  • “Cmd-End”: “goDocEnd”,
  • “Cmd-Down”: “goDocEnd”,
  • “Cmd-U”: “Scroll Up (or PageUp key)”,
  • “Cmd-D”: “Scroll Down (or PageDown key)”,
  • “Alt-Left”: “goWordLeft”,
  • “Alt-Right”: “goWordRight”,
  • “Cmd-Left”: “goLineStart” (or Ctrl-A),
  • “Cmd-Right”: “goLineEnd” (or Ctrl-E)

Delete:

  • “Alt-Backspace”: “delWordBefore”,
  • “Ctrl-Alt-Backspace”: “delWordAfter”,
  • “Alt-Delete”: “delWordAfter”,
  • “Ctrl-U”: “delete from beginning of line to cursor”,
  • “Ctrl-K”: “delete from cursor to the end of line”,

File commands:

  • “Cmd-O”: “Open”,
  • “Cmd-N”: “New”,
  • “Cmd-S”: “Save”,
  • “Shift-Cmd-S”: “SaveAs…”,

Find and replace:

  • “Cmd-F”: “find”,
  • “Cmd-G”: “findNext”,
  • “Shift-Cmd-G”: “findPrev”,
  • “Cmd-Alt-F”: “replace”,
  • “Shift-Cmd-Alt-F”: “replaceAll”,

Indent line(s) of code:

  • “Tab”: “indentMore”
  • “Shift-Tab”: “indentLess”,

Function specific to C-like modes

When in either Javascript, java or C modes
implements type-ahead feature on entering if, then, else, switch, for, while and do-while keywords.

HTML specific Emmet command:

Formerly known as Zen Coding abbreviate language.

  • “Cmd-E”: “Expand abbreviate”,
  • “Tab”: “Expand abbrev” as Cmd-E does

Markdown dedicated formatting commands

  • “Cmd-B”: “Bold”,
  • “Cmd-I”: “Italic”,
  • “Cmd-J”: “Code” (surround selection with backticks),
  • “Cmd-U”: “Underline” (only applies under Haroopad),
  • “Cmd-Alt-J”: “Cite” block of lines,
  • “Cmd-K”: “Web link”,
  • “Cmd-L”: “Bullet list” (ul) block of lines
  • “Cmd-Alt-L”: “Numbered list” (ol) block of lines,

Also available as popup context menu entries.

Some functions like jshint are automatically activated when the current file type is Javascript. Shortcut [Ctrl-SPACE] triggers code completion based on the words found around in the present document (not the language vocabulary).
