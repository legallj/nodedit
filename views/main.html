nodEdit — Minimal IDE

2014-03-20

Minimal IDE dedicated to single page application (SPA) development under NodeJS with WebKit. Implement the read - execute - print - loop (REPL) paradigme with HTML5, CSS3, Javascript and Markdown files.

Built in keyboard commands

Extract from Source code

  • “Cmd-C”: “Copy”,
  • “Cmd-X”: “Cut”,
  • “Cmd-V”: “Paste”,
  • “Cmd-A”: “selectAll”,
  • “Cmd-D”: “deleteLine”,
  • “Cmd-Z”: “Undo”,
  • “Shift-Cmd-Z”: “Redo”,
  • “Cmd-Y”: “Redo”, (yank)
  • “Ctrl-Space”: “autocomplete”, (agnostic)

Move cursor:

  • “Cmd-Up”: “goDocStart”,
  • “Cmd-End”: “goDocEnd”,
  • “Cmd-Down”: “goDocEnd”,
  • “Alt-Left”: “goWordLeft”,
  • “Alt-Right”: “goWordRight”,
  • “Cmd-Left”: “goLineStart”,
  • “Cmd-Right”: “goLineEnd”,

Delete:

  • “Alt-Backspace”: “delWordBefore”,
  • “Ctrl-Alt-Backspace”: “delWordAfter”,
  • “Alt-Delete”: “delWordAfter”,

File commands:

  • “Cmd-O”: “Open”,
  • “Cmd-N”: “New”,
  • “Cmd-S”: “Save”,
  • “Shift-Cmd-S”: “SaveAs...”,

Find and replace:

  • “Cmd-F”: “find”,
  • “Cmd-G”: “findNext”,
  • “Shift-Cmd-G”: “findPrev”,
  • “Cmd-Alt-F”: “replace”,
  • “Shift-Cmd-Alt-F”: “replaceAll”,

Indent line:

  • “Cmd-[”: “indentLess”,
  • “Cmd-]”: “indentMore”
